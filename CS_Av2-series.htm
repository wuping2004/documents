We'll retire Basic and Standard A-series VMs.

Migrate your workloads from Basic and Standard A-series VMs to Av2-series VMs before 31 August 2024

You're receiving this email because you use Basic or Standard A-series virtual machines (VMs).

On 31 August 2024, we'll retire Basic and Standard A-series VMs. Before that date, you'll need to migrate your workloads to Av2-series VMs, which provide more memory per vCPU and faster storage on solid-state drives (SSDs).

Required action

To avoid service disruptions, migrate your workloads from Basic and Standard A-series VMs to Av2-series VMs before 31 August 2024.

Help and support

If you have questions, ask community experts in Microsoft Q&A. If you have a support plan and need technical help, create a support request:

  1. For Issue type, select Technical.
  2. For Subscription, select your subscription.
  3. For Service, click My services.
  4. For Service type, select Virtual Machine running Windows/Linux.
  5. For Summary, enter a summary of your request.
  6. For Problem type, select Assistance with resizing my VM.
  7. For Problem subtype, select the option that applies to you.
